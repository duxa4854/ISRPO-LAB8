# Лабораторная работа: Сравнение Flexbox и CSS Grid

## Описание работы

В ходе данной лабораторной работы проводится сравнительный анализ двух современных методов верстки в CSS — **Flexbox** и **CSS Grid**. Цель — понять их принципы работы, различия в подходах к разметке, а также определить оптимальные сценарии применения каждого из них.

Работа включает:
- Изучение теоретических основ Flexbox и CSS Grid;
- Анализ ключевых критериев (тип компоновки, управление, назначение и т.д.);
- Практическое применение в типовых задачах верстки;
- Формулировку выводов и рекомендаций.

---

## Сравнение Flexbox и CSS Grid

### Общая идея

| Критерий            | Flexbox                     | CSS Grid                     |
|---------------------|-----------------------------|------------------------------|
| Тип компоновки      | Одномерная                  | Двумерная                    |
| Управление          | Строка *или* столбец        | Строки *и* столбцы           |
| Основное назначение | Выравнивание элементов      | Построение макетов           |
| Подходит для        | Компонентов                 | Страниц и сеток              |

**Вывод**: Flexbox лучше подходит для **линейных структур**, Grid — для **табличных и сложных** макетов.

## Типичные сценарии использования

| Задача                      | Рекомендуется       |
|----------------------------|---------------------|
| Навигационное меню         | Flexbox             |
| Карточки товаров            | Flexbox + Grid      |
| Центрирование элемента     | Flexbox             |
| Макет страницы             | Grid                |
| Галерея изображений        | Grid                |
| Дашборд                     | Grid                |

>Примечание: В реальных проектах часто используется **гибридный подход** — например, Grid для общей структуры страницы и Flexbox внутри отдельных секций (например, для выравнивания кнопок или карточек).

## Заключение

- **Flexbox** — мощный инструмент для управления распределением пространства в одном измерении. Идеален для компонентов, где важна гибкость и адаптивность.
- **CSS Grid** — универсальный инструмент для создания сложных, предсказуемых макетов. Особенно эффективен при работе с фиксированными или относительными размерами ячеек.

Выбор между ними зависит от задачи:  
* Если нужно выстроить *ряд элементов* — используйте Flexbox.  
* Если нужно спроектировать *сетку из строк и столбцов* — выбирайте Grid.
